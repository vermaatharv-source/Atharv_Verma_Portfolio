## Overview

This is my personal developer portfolio — designed and coded from scratch with **no frameworks, no templates, no page builders**. Just HTML5, CSS3, and vanilla JavaScript, deployed on GitHub Pages.

The whole thing is styled around a dark, terminal/code aesthetic: macOS-style terminal windows, a green-on-dark color scheme, monospace typography, and a typing animation in the hero — built to actually feel like a developer's workspace rather than a generic "About Me" page.

## ✨ Features

- **Single-page scroll experience** — smooth-scroll navigation across six sections
- **Terminal-window UI motif** — consistent signature component used throughout (hero, about, resume, contact)
- **Live typing effect** in the hero, cycling through rotating taglines
- **Fully responsive** — mobile nav toggle, breakpoints down to small phone screens
- **Embedded, downloadable resume** — PDF viewer built directly into the page, no redirect needed
- **Scroll-reveal animations** on section and card entry, using `IntersectionObserver`
- **Accessible by default** — visible keyboard focus states, `prefers-reduced-motion` respected
- **Zero dependencies** — no build step, no npm install, just open `index.html`

## 🚀 Run Locally

```bash
git clone https://github.com/vermaatharv-source/Atharv_Verma_Portfolio.git
cd Atharv_Verma_Portfolio
```

Then just open `index.html` in your browser — or serve it locally:

```bash
# Python
python -m http.server 8000

# Node
npx serve
```
